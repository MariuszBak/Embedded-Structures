# Embedded Structures

Repozytorium zawiera zestaw struktur danych oraz narzędzi pomocniczych dla projektów **embedded**, przygotowanych głównie pod język **C**.

## Aktualna zawartość

Obecnie projekt zawiera:

- struktury danych dla **fontów**
- struktury danych dla **obrazów**
- opisy formatów używanych w języku **C**

## Aplikacje

Repozytorium zawiera również narzędzia webowe (HTML):

- edytor fontów
- edytor obrazów

Aplikacje będą działały lokalnie w przeglądarce jako proste utility tools.

## Zastosowanie

Projekt może być używany do:

- systemów embedded
- mikrokontrolerów
- interfejsów graficznych LCD / OLED / TFT
- konwersji zasobów graficznych do struktur C

## Status projektu

Projekt jest w trakcie rozwoju. Kolejne funkcje będą dodawane sukcesywnie.
